## 📖 Overview

This repository contains a collection of Arduino-based IoT and Embedded Systems projects developed for learning, experimentation, and practical implementation of sensor-based automation systems.

These projects demonstrate the integration of sensors, actuators, microcontrollers, and real-time control mechanisms using Arduino.

---

## 🛠️ Technologies & Components Used

* Arduino UNO
* Arduino IDE
* Embedded C / Arduino Programming
* HC-SR04 Ultrasonic Sensor
* IR Flame Sensor
* Joystick Module
* Push Buttons
* Servo Motor
* Buzzer
* LEDs
* Breadboard
* Jumper Wires

# 🎮 Project 1 – Joystick Controller LED

## Description

A joystick-controlled LED system that uses analog input values from the joystick module to control LED behavior based on directional movement.

## Features

* X and Y Axis Detection
* Analog Input Processing
* LED Direction Control
* Real-Time Response

## Skills Demonstrated

* Analog Signal Reading
* Arduino GPIO Programming
* Hardware Interfacing

# 🗑️ Project 4 – Smart Dustbin / Smart Box

## Description

A touchless smart dustbin system that automatically opens its lid when a hand or object is detected nearby using an ultrasonic sensor.

## Features

* Automatic Lid Opening
* Contactless Operation
* Ultrasonic Distance Detection
* Servo Motor Control

## Components
