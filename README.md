# 🚀 IoT Projects using Arduino

A collection of Arduino-based IoT and Embedded Systems projects demonstrating sensor integration, automation, control systems, and real-time hardware interaction.

## 📋 Overview

This repository contains multiple Arduino projects developed for learning and demonstrating IoT concepts, embedded programming, and hardware interfacing.

## 🛠️ Technologies Used

- Arduino UNO
- Arduino IDE
- Ultrasonic Sensor (HC-SR04)
- Joystick Module
- Push Buttons
- Buzzer
- Servo Motor
- LEDs
- Breadboard & Jumper Wires
- Embedded C / Arduino Programming

---

# 🎮 Project 1 - Joystick Controller LED

## Description

This project uses a joystick module to control LEDs based on directional movement. It demonstrates analog input processing and digital output control using Arduino.

## Features

- X-axis and Y-axis movement detection
- LED control using joystick directions
- Real-time response
- Analog signal processing

## Learning Outcomes

- Analog Input Reading
- Arduino Pin Configuration
- LED Control Logic
- Hardware Interfacing

# 📡 Project 3 - Radar System with Ultrasonic Sensor

## Description

An Arduino-based radar system that uses an ultrasonic sensor and servo motor to detect nearby objects and display scanning movement.

## Features

- Object Detection
- Distance Measurement
- Servo Motor Rotation
- Radar-Style Scanning

## Components

- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- Breadboard

## Learning Outcomes

- Distance Measurement
- Sensor Integration
- Servo Control
- Real-Time Data Processing
